# 饿汉模式
为防止被手动初始化, 一般单例类型不可导出, 需要以可导出接口的形式返回
# 懒汉模式
```go
func GetInstance() Instance {  
    if s != nil {  
        return s  
    }  
    mux.Lock()  
    defer mux.Unlock()  
    if s != nil {  // double check
        return s  
    }  
    s = newSingleton()  
  
    return s  
}
```
规范实现: `sync.Once.Do()`
