11.1-2
位向量的每一位代表是否包含这个位置索引的元素
```
BITMAP-SEARCH(V, k)
    if V[k] != 0
        return k
    else return NIL
```
```
BITMAP-INSERT(V, x)
    V[x] = 1
```
```
BITMAP-DELETE(V, x)
    V[x] = 0
```
11.2-4
必须要双向链表
搜索: 不变, 复杂度$O(1)$
插入: 如果目标槽位被占用, 从空闲列表头部取出一个空闲槽位, 将新元素存入这个槽位并插入到`T[h(x.key)]` 的链表头部; 如果未占用, 存入槽位后要从空闲列表移除槽位, 需要用到双向链表,复杂度$O(1)$
删除: 从链表断开, 归还槽位, 复杂度$O(1)$
